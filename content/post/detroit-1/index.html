Using the Detroit Open Data portal, prepare the Parcels and Sales data sets for analysis. Create a basic descriptive RMarkdown report which analyzes relevant trends and data quality issues. Contextualize your analysis with key literature on properties in Detroit.

Construct a classification problem for properties in Detroit and undertake an initial analysis. I suggest that you add an additional data set from the open data portal or another source. Searching for property data sets in the open data portal includes many property attributes which could be analyzed including permits/violations/registrations/rental status/auctions/government owned property.

Assignment

Instead of traditional problem sets, this course has a single four part assignment where you will build upon your previous work each week with new material from the course. You will explore property assessment in Detroit, Michigan and create an assessment model. After the completion of the assignment, you will wrap your model into a report which analyzes the effectiveness of your model based on the ethical and other frameworks from class and make a brief presentation to the class.

Grading

20% - completes all four assignments by deadline (5% per week)

15% - timely incorporation of feedback and new concepts from course (5% each for assignments 2-4)

25% - final code

15% - final report

25% - final presentation
